				==>>Node.js Assignment<<==

This is a simple nodejs login and register functionality using Passport.js and MongoDB. Express.js is also used to create server.

Steps to run : 
->npm install
->npm start

//To run with Nodemon
->npm run dev

//To open on web browser
->localhost:8080


//To connect to MongoDB
Open "config/keys.js" and add your MongoDB URI, local or Atlas.
